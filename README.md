# Stenciljs and Webcomponents

Talk held at Hochschule Augsburg on 13. June 2019
by Sebastian Oley, Webworker United GmbH

This repo includes slide and examples. You will need [Node.js](http://nodejs.org/) to run these.

## Slides

```sh
$ cd slides
$ npm i
$ npm start
```

Open <http://localhost:8000> to view the presentation.

See Readme for more info.

## Examples

```sh
$ cd examples/<example>
$ npm i
$ npm start
```

See corresponding README files for more info.
